Steps to execute the program

The code is developed in the Jupyter notebook using Python.

Libraries to be installed 

1. Install Python https://www.python.org/downloads/
2. Execute the below command to install libraries
	- pip install NLTK or download the libraries from the link https://www.nltk.org/install.html
	- import nltk
	- nltk.download() # This will install all the corpus and data files.
	
3. Install Jupyter Notebook
4. Import the file to jupyter notebook and run all cells.

Note - The DRT output will be printed on a seprate window, make sure to close it before you execute the next cell.
